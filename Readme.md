# IOT Data Collector with Mqtt Broker

#### The Project consists of :

- Backend: Spring Boot
- Frontend: React JS
- Database : Postgres
- MQTT : mosquitto


### How to run the project on docker :


#### clone the project: 

```bash
  git clone 
```
#### Access directory

```bash
  cd 
```

#### Run Docker

```bash
  docker compose up --build -d
```
#### Access FrontEnd application
```bash
  localhost:3000
```
